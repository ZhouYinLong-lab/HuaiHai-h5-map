## 功能覆盖

- 地图探索：8 节点、拖拽、滚轮/双指缩放、重置、节点自动聚焦。
- 遗址名录：名称/地区/类型搜索，省份筛选，档案详情，地图定位。
- 项目说明：项目定位、内容边界、技术方式、动态内容准备度、素材清单。
- 遗址详情：封面、摘要、地址、历史正文、内容状态、图集槽位、来源槽位、视频和高德跳转。
- 多端导航：手机/平板底栏，桌面档案侧栏。
- 发布能力：Web App Manifest、离线应用外壳、Azure Static Web Apps 配置。
