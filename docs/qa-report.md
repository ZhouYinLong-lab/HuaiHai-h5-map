# 地标事件地图重构 QA 报告

| 项目 | 值 |
|---|---|
| 审查日期 | 2026-06-25 |
| 项目名 | 淮海战役红色遗址交互地图 |
| 构建环境 | Node 20.19.5 / Vite 8.1.0 |
| 本轮范围 | 地标扩充、事件叙事、原创底图和地图交互重构 |

## 本轮成果

- 地标由 8 处扩展至 16 处。
- 覆盖 5 个战场与战斗点、6 个指挥旧址、1 个起义事件点、4 个纪念设施。
- 第一阶段 4 处、第二阶段 6 处、第三阶段 3 处、纪念传承 3 处。
- 新增事件日期、事件标题、地标类型、标签方向和资料来源字段。
- 新增战役阶段筛选、地标类型图例和名录阶段筛选。
- 新增三阶段战役时间线，相关地标可直接跳转并聚焦地图。
- 原底图替换为原创 SVG 态势图，包含河流、铁路、城镇、阶段战区和作战动线。
- 手机和平板总览只显示地标类型符号，阶段筛选后显示完整标签；桌面总览显示名称。

时间线回归：

- 3 个战役阶段均正常渲染。
- 共关联 13 处阶段地标，另有 3 处纪念传承节点保留在地图和名录。

- 点击时间线地标可切回地图、聚焦对应节点并打开详情。
- 手机底栏、平板底栏与桌面侧栏均显示 4 个导航入口。
