# 项目资料收集清单

本清单用于收集最终上线所需的历史资料、图片、视频、地图和版权信息。建议每项资料都同时保存原文件、来源链接和授权说明。

## 一、全项目公共资料

### 1. 正式地图底图

- [ ] 淮海战役作战图或区域态势图高清文件
- [ ] 图片来源或收藏单位
- [ ] 作者、出版物名称及出版信息
- [ ] 公开使用依据或书面授权
- [ ] 可否裁剪、调色和添加交互标记的说明
- [ ] 推荐分辨率不低于 2400 × 1600
- [ ] 最终网页版本优先使用 WebP 或 AVIF

> 如后续更换底图，需要重新校准 16 个节点的百分比坐标。

### 2. 项目介绍

- [ ] 社会实践项目正式名称
- [ ] 项目背景，建议 300–500 字
- [ ] 实践目标，建议 150–300 字
- [ ] 实践过程及调研方式
- [ ] 项目成果总结
- [ ] 指导教师、学校、学院和团队署名
- [ ] 项目完成日期
- [ ] 联系方式或负责部门（如需公开）

### 3. 统一视觉素材

- [ ] 学校 Logo 及使用规范
- [ ] 学院或实践团队 Logo
- [ ] 项目二维码
- [ ] 分享封面图，建议 1200 × 630
- [ ] 网站图标或项目标识
- [ ] 片头、片尾或统一题字素材（如有）

## 二、每处遗址都需要的资料

以下内容应为每处遗址分别准备一份。

### 1. 基础信息

- [ ] 遗址正式全称
- [ ] 地图短名称
- [ ] 所在省、市、县区
- [ ] 详细地址
- [ ] 高德地图中的准确地点名称
- [ ] 遗址类型：纪念馆、烈士陵园、旧址等
- [ ] 所属战役阶段或历史主题
- [ ] 开放时间和参观要求（如需展示）
- [ ] 联系电话（如需展示）

### 2. 历史文字

- [ ] 80–150 字摘要
- [ ] 300–600 字历史正文
- [ ] 重要时间节点
- [ ] 相关历史人物
- [ ] 遗址沿革
- [ ] 纪念意义或展陈特色
- [ ] 每一项重要事实的来源

### 3. 图片

- [ ] 1 张遗址封面图
- [ ] 2–5 张历史照片或现场照片
- [ ] 每张图片的图注
- [ ] 拍摄时间或历史年代
- [ ] 摄影者、收藏单位或出处
- [ ] 版权状态和使用授权
- [ ] 图片无水印版本

建议规格：

| 用途 | 建议尺寸 | 建议格式 | 文件大小 |
|---|---:|---|---:|
| 封面 | 1600 × 1000 | WebP | 300KB 内 |
| 图集 | 长边 1600px | WebP | 单张 400KB 内 |
| 老照片扫描 | 长边不少于 1800px | TIFF/JPEG 原件 + WebP 网页版 | 保留原文件 |

### 4. 视频

- [ ] B站或权威平台的正式视频链接
- [ ] 视频标题
- [ ] 发布单位
- [ ] 发布时间
- [ ] 视频封面图（可选）
- [ ] 是否允许外链展示
- [ ] 链接稳定性检查

视频不需要下载或上传到本项目。

### 5. 权威来源

每处遗址建议至少有 1–3 个来源：

- [ ] 纪念馆或遗址官方页面
- [ ] 当地政府或文旅部门页面
- [ ] 党史、地方志或档案馆资料
- [ ] 正式出版的书籍、论文或图册
- [ ] 来源标题、发布单位、出版时间和链接
